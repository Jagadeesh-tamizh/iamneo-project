## Run Locally

Clone the project

```bash
  git clone gh repo clone Jagadeesh-tamizh/iamneo-project
```

Go to the project directory

```bash
     cd iamneo-project-main
```

Install dependencies

```bash
    npm install
```

Make build package
```
    npm run build
```
Start the server

```bash
    npm install -g serve
    serve -s build
```
